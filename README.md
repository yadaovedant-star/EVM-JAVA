🗳️ EVM Java Project

📌 Description

This is a simple Electronic Voting Machine (EVM) simulation built using Java AWT.

🚀 Features

- Vote for 4 candidates
- Close voting system
- Display results in percentage
- Reset voting system

▶️ How to Run

1. Download all files
2. Open terminal in the folder
3. Compile:
   javac EVM.java
4. Run:
   java EVM

🖼️ Note

Make sure all image files (candA.jpeg, candB.jpeg, etc.) are in the same folder.
